# Project Summary

I built the worlds best to do list manager...for me. 

It's all keyboard accessible with a help menu for everyone who can't remember the short cuts (which includes me).

Could I have built this in neovim? Yes. Should I have built this in neovim? probably. Will I build it in neovim? maybe next week.

Every single dev and their dog has a to do list app on their github. I was missing one. And honestly I thought I didn't need it for my portfolio. Turns out I did need it to manage my portfolio...but mostly to manage all the stuff I'm doing in my portfolio. 

Features I think are genuinely unique: 
- I tied the ^ key to sort ascending/descending.
- Filter call outs in the top (or wherever I put it)
- I think my themes are pretty cool
