tdx_ // color themes
Six preselected phosphor palettes for the terminal to-do. Each one re-skins the entire UI by re-declaring the token set — same chrome, same density, different glow. Neon terminal green is in.
6 palettesmonochrome accent + semantic green / red / cyandrop-in via themes.css
How it wires up. Set data-theme="green" on a root element (e.g. <html>) and every var(--…) token cascades to the new palette — no markup changes. No attribute falls back to amber. Semantic colors (success / overdue / recurrence) are tuned per theme so they never collide with the accent.
